AUTRES PARAMETRES POUR UNE TACHE ETAPE 1C

Si l'on reprend les données JSON de l'étape 1A, on peut retenir deux critères et écrire un jeu de valeur pour chacun d'eux :

  • Le critère "etiquette" avec les valeurs Perso ou Pro
  • Le critère "statut" avec les valeurs A faire, En cours, Terminée ou Supprimée

Ecriture du jeu de valeur du critère "etiquette"

Un label et une description ont été ajoutés à ce jeu de valeur.

Structure JSON déployée

        {
        "etiquette": [
            {
            "idEtiquette": 1,
            "label": "Perso",
            "description": "Tâches personnelles, liées à la vie quotidienne ou aux loisirs."
            },
            {
            "idEtiquette": 2,
            "label": "Pro",
            "description": "Tâches professionnelles, liées au travail ou aux projets."
            }
            ]
        }

Structure JSON compactée

        {"etiquette":[{"idEtiquette":1,"label":"Perso","description":"Tâches personnelles, liées à la vie quotidienne ou
        aux loisirs."},{"idEtiquette":2,"label":"Pro","description":"Tâches professionnelles, liées au travail ou aux
        projets."}]}

Un label et une description ont été ajoutés à ce jeu de valeur.

    {

        "description": "Tâche à réaliser, pas encore commencée."
